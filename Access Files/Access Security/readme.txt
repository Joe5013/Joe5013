Introduction
------------

The Secfaq.exe file contains a Microsoft Word document titled 
"Frequently Asked Questions About Microsoft® Access Security for 
Microsoft Access versions 2.0 through 2000." As the title states, 
it answers just about any Microsoft Access security question you may have.
